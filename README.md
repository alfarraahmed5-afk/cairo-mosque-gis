# Cairo Mosque GIS

A web-based GIS project that maps mosques across Cairo and provides navigation to reach them using nearby metro stations. The project combines manually collected geospatial data with a lightweight web application to create a gamified experience for visiting a different mosque each night during Ramadan.

## Project Overview

Cairo Mosque GIS was created as part of a personal Ramadan challenge: visiting and praying at a different mosque every night for 30 nights. The project integrates geospatial mosque data with Cairo metro station locations and provides route guidance through a web-based interface.

The application also tracks progress throughout Ramadan, turning the experience into a simple gamified journey across Cairo's mosques.

## Key Features

- Interactive map displaying mosque locations across Cairo
- Integration of nearby Cairo Metro stations for navigation
- Route generation using mapping services
- Gamified progress tracker for mosque visits during Ramadan
- Lightweight web-based interface (HTML + JavaScript)
- KML-based geospatial data structure

## Data Collection

All mosque locations and metro stations were manually collected and organized into a KML dataset. This dataset acts as the primary geographic data layer used by the application.

The data includes:
- Mosque name
- Geographic coordinates
- Metro station proximity

## Technical Stack

- HTML
- JavaScript
- KML (Geospatial data format)
- Web mapping APIs for route generation
- AI-assisted coding for application development

## How It Works

1. The KML dataset loads mosque and metro station locations.
2. The web application visualizes the data on an interactive map.
3. The user selects a mosque to visit for the night.
4. The app generates directions using nearby metro stations.
5. Progress is recorded as part of the Ramadan challenge.

## Use Case

The project demonstrates how simple GIS datasets can be combined with web technologies to create interactive geospatial tools. It also showcases practical applications of manually curated geographic data and lightweight web-based mapping systems.
